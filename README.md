# Video-Game
![image](./screenshot.png)

This is Snake game. The Game consists a leaderboard where the other players score shown as well,You can set the speed of the snale and then start the game.
## Built With

- Html
- Phaser
- Javascript
- webpack

## How to Run
- npm install --save-dev
- npm run dev
- http://localhost/8080
